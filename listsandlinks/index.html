Instructions to Create a New Webpage on the Production Server

  1. Create a 2 file in your repository by typing "gedit" in your terminal and enter
  2. The first one is index.html and another one is main.css
  3. When the file is open type in the basic code of html
  4. You can type in the name of your new webpage in the title section of yur html code, and type in your header in the header section
  5. To see your new webpage, you have to add your html file into github first.
  6. To add you file into github, you do the git process thing, where you type "git add" and then your file name and "git commit" and lastly, git push, then your type in your username and password.
  7. In your terminal type "python -m SimpleHTTPServer" and enter.
  8. Open new tab and type "localhost:8000" and you should see your new webpage.
  9. If you want to edit your texts, like change your font color or size, you can do it in your main.css file. For example, if you want to change the size of your text then type in "font-size:".
